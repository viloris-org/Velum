# Support Policy

Velum currently has no supported versions, deployment SLA, compatibility
window, or production operating commitment.

Public GitHub issues may be used for reproducible, non-sensitive defects after
the repository is hosted. Questions and proposals should identify the roadmap
item or evidence gate they affect. Security-sensitive material must follow
[`SECURITY.md`](SECURITY.md) and must not be filed publicly.

Support commitments, supported platforms, and response targets will be
published only when Stage 6 readiness evidence exists.
